Les listes maîtresses sont des listes standard d'articles qui ne peuvent être définies qu'au niveau du serveur central.

## Consulter les Listes Maîtresses

Allez à `Catalogue` > `Listes Maîtresses` dans le panneau de navigation à gauche :

![Barre de navigation montrant les Listes Maîtresses en surbrillance sous Catalogue](images-en/cat_gotoml.png)

Vous verrez une fenêtre comme celle-ci :

![Page des Listes Maîtresses affichant les listes disponibles](images-en/cat_mllist.png)

Pour afficher tous les articles d'une liste maîtresse, cliquez simplement sur l'une des listes qui s'affichent à l'écran :

![Détail d'une liste maîtresse montrant les articles avec codes, noms et unités](images-en/cat_mldetail.png)
